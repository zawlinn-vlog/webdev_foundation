### Naming Convention for Variables

- Allowed Characters: Names can contain letters (A-Z, a-z), digits (0-9), underscores (\_), and dollar signs ($).

- Starting Character: A variable name must begin with a letter, an underscore, or a dollar sign; it cannot start with a digit.

- Case Sensitivity: JavaScript variable names are case-sensitive (dogName and DogName are different variables).

- Reserved Words: You cannot use JavaScript keywords or reserved words (like let, const, function, if) as variable names.

- No Spaces/Hyphens: Variable names cannot contain spaces or hyphens (-). Hyphens are often mistaken as subtraction attempts.
