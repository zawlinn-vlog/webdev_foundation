## CSS3 - Cascading Style Sheet v3

1. Internal Style (in HTML file)

```html
<style>
  selector {
    property: value;
  }
</style>
```

2. Inline Style (in HTML file)

```html
<h1 style="color:tomato">This is Heading</h1>
```

3. External Style (CSS file)

```html
<link rel="stylesheet" href="./assets/css/main.css" type="text/css" />

<!-- type="text/css" is optional -->
```

## CSS Syntax &mdash;
