## CSS3 - Margin/Padding

```scss
.main {
  // Margin

  margin-top: 1rem;
  margin-bottom: 1rem;
  margin-left: 1rem;
  margin-right: 1rem;
  // margin: top right bottom left;
  // margin: top left/right bottom;

  // margin:top/bottom/left/right;

  padding-right: 1rem;

  padding-left: 1rem;
```
