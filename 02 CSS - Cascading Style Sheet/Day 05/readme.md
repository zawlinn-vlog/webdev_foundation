## Transition &mdash;

- Transition-property (Required)
- Transition-duration (Required)
- Transition-timing-function
- Transition-dalay

### Transition Speed Curve &mdash;

- ease &mdash; Transition will start slow, then go fast and end slow (Default)
- linear &mdash; Transition will keep the same speed from start to end
- ease-in &mdash; Transition will start slow
- ease-out &mdash; Transition will end slow
- ease-in-out &mdash; Transition will have a slow start and end

## Shortcut

```scss
.box {
  .rotate_box {
    transition-property: transform;
    transition-duration: 0.3s;
    transition-timing-function: linear; // ease is default, linear, ease-in, ease-out, ease-in-out, cubic-bezier(n,n,n,n)

    // transition: property duration timing-function delay;
  }

  &:hover .rotate_box {
    transform: rotate(2turn);
  }
}
```
